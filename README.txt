### MPI+OpenMP-реализация

Исключением стал только случай с MPI=8 и OpenMP=8. Это особая ситуация, где каждому процессу выделялось 4 ядра, и на каждом ядре работало по два потока.
Во всех остальных случаях каждому процессу выделялось количество ядер, строго соответствующее числу потоков.
При этом MPI-процессы распределялись по вычислительным узлам равномерно.
